## Archived Memory Examples

This folder contains committed example files that mirror the runtime layout of the archived memory workspace.

Included example date:
- `daily_sessions/2026-05-20/2026-05-20.md` for the archived parent daily note
- `daily_sessions/2026-05-20/2026-05-20-0001.md` for the archived session transcript
- `daily_sessions/summaries/2026-05-20.summary.md` for the archived daily summary

These files document the expected archive structure without exposing real user memory data.
